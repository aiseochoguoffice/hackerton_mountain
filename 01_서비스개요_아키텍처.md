# 🏔️ 산행 안전 서비스 — 서비스 개요 & 아키텍처

> **산악 활동 위험 지수 기반 사전 체크 & 정보 제공 플랫폼**  
> 소방청 전국 산악사고 데이터(2020, 2024)를 기반으로 산별 위험 지수·사고 유형·유의사항을 제공하고, 산행 전 스스로 위험 요소를 체크하는 서비스

---

## 1. 서비스 컨셉

```
[산행 전]                [산행 중]              [사고 후]
위험지수 확인 →   →   실시간 날씨 경보   →   사고 통계 리포트
산별 체크리스트   →   위치 공유(선택)    →   데이터 축적
사고 유형 학습    →   긴급 신고 바로가기  →
```

### 핵심 기능 4가지
| # | 기능 | 설명 |
|---|------|------|
| 1 | **산별 위험 지수** | 사고 빈도·유형·계절·시간대 기반 위험 지수(1~100) |
| 2 | **사고 유형 탐색** | 실족추락/조난/개인질환/기타 유형별 유의사항 |
| 3 | **사전 체크리스트** | 산행 전 장비·체력·날씨·동행 등 자가 점검 |
| 4 | **산별 특성 카드** | 산 이름 검색 → 난이도·사고다발지점·주의사항 |

## 2. 데이터 소스 분석 (소방청 공공데이터)

### 현황 파일 (`소방청_전국_산악사고_현황_20241231.csv`)
| 컬럼 | 설명 | 활용 |
|------|------|------|
| 신고년월일 / 신고시각 | 사고 발생 일시 | 시간대·계절별 위험 지수 |
| 발생장소_시/구/동/리 | 지역 정보 | 지역별·산별 사고 매핑 |
| 사고원인 | 산악/기타 | 분류 필터 |
| 사고원인코드명_사고종별 | 사고부상_실족, 조난 등 | **사고 유형 분류** |
| 처리결과코드 | 구급대인계/항공대인계/인명구조 | 중증도 지표 |
| 구조인원 | 구조된 인원 수 | 규모 지표 |

### 구조활동 파일 (`소방청_전국_산악사고_구조활동현황_20201231.csv`)
| 컬럼 | 설명 | 활용 |
|------|------|------|
| 출동년월일 / 출동시각 | 출동 일시 | 대응 시간 계산 |
| 번지 (산 번지) | 산1-1 형태 | 산 위치 식별 |
| 사고원인 | 실족추락/개인질환/조난/기타산악 | **사고 유형 분류** |

### 사고 유형 분류 체계
```
실족추락 (가장 빈번)
  ├── 미끄러짐 (우천·결빙)
  └── 추락 (급경사·야간)

개인질환
  ├── 심혈관계
  └── 탈진·저체온

일반조난
  ├── 길 잃음
  └── 고립 (악천후)

기타산악
  ├── 낙석
  ├── 벌쏘임
  └── 장비 부족
```

       │               │

## 4. 페이지 구성 (와이어프레임 개요)

```
/ (홈)
├── /mountains              # 산 목록 & 검색
│   └── /mountains/:id      # 산별 상세 (위험지수, 사고통계, 유의사항)
├── /checklist              # 산행 전 체크리스트
│   └── /checklist/result   # 체크 결과 & 맞춤 조언
├── /risk-map               # 전국/서울 위험 지수 지도
├── /accident-types         # 사고 유형 가이드
└── /stats                  # 사고 통계 대시보드
```

## 5. 위험 지수 계산 로직

```
위험지수(0~100) = 
  사고빈도_가중치(40%) 
  + 중증도_가중치(30%)       ← 항공대인계 > 구급대인계 > 인명구조
  + 계절_보정(15%)           ← 봄 단풍/겨울 결빙 시즌 가중
  + 시간대_보정(15%)         ← 14~16시 하산 시 최고조

등급: 낮음(0~30) / 보통(31~60) / 높음(61~80) / 매우높음(81~100)
```
